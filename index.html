eGlobe UI System

A theme-independent, shared UI library for all eGlobe Shopify Apps.

v1.3.1 Responsive

Getting Started

Follow these simple steps to integrate eGlobe UI into your project.

1. Link CSS Library

Include the main entry CSS file in your <head>.

<link rel="stylesheet" href="path/to/eg-ui.css">
2. Interactive Components

For interactive features like Modals, Tabs, or Toasts, include the library JS file.

<script src="path/to/eg-ui.js"></script>

Modals

Dialog prompts for critical information or complex interactions.

How to Use (Code)

1. Define the Modal HTML structure with a unique ID.

<div class="eg-ui-modal" id="myModal">
  <div class="eg-ui-modal__overlay"></div>
  <div class="eg-ui-modal__content">
    <div class="eg-ui-modal__header">
      <h3 class="eg-ui-modal__title">Title</h3>
      <button class="eg-ui-modal__close" onclick="closeModal('myModal')">×</button>
    </div>
    <div class="eg-ui-modal__body">Content goes here.</div>
  </div>
</div>

2. Trigger the modal using JavaScript:

openModal('myModal'); // To open
closeModal('myModal'); // To close

Popovers

Small overlays of content that are used to explicate on specific elements.

How to Use (Code)

Use data-eg-ui-toggle="popover" and data-target to link a trigger to a popover.

<!-- Popover Trigger -->
<button class="eg-ui-btn eg-ui-btn--primary" 
        data-eg-ui-toggle="popover" 
        data-target="popoverId">
  Click me
</button>

<!-- Popover Content -->
<div class="eg-ui-popover eg-ui-popover--top" id="popoverId">
  <div class="eg-ui-popover__arrow"></div>
  <h4 class="eg-ui-popover__header">Popover Title</h4>
  <div class="eg-ui-popover__body">
    And here's some amazing content. It's very engaging. Right?
  </div>
</div>

Toasts

Temporary notifications that slide into view.

How to Use (Code)

Display notifications programmatically using the egToast object.

egToast.show({
  message: 'Action successful!',
  type: 'success', // 'success' | 'info' | 'warning' | 'danger'
  position: 'top-right', // 'top-left' | 'top-center' | 'top-right' | 'bottom-left' | 'bottom-center' | 'bottom-right'
  duration: 3000,
  showClose: false
});

Utilities Cheatsheet

Layout classes have responsive md: and lg: variants (e.g., md:eg-ui-flex, lg:eg-ui-grid-cols-2).

Display & Position

Class Properties
.eg-ui-block display: block;
.eg-ui-inline-block display: inline-block;
.eg-ui-flex display: flex;
.eg-ui-grid display: grid;
.eg-ui-hidden display: none;
.eg-ui-relative, .eg-ui-absolute, .eg-ui-fixed, .eg-ui-sticky position: relative/absolute/fixed/sticky;

Flexbox & Grid

Class Properties
.eg-ui-flex-row, .eg-ui-flex-col flex-direction: row / column;
.eg-ui-flex-wrap, .eg-ui-flex-nowrap flex-wrap: wrap / nowrap;
.eg-ui-items-start|center|end|stretch align-items: flex-start|center|flex-end|stretch;
.eg-ui-justify-start|center|end|between|around justify-content: ...;
.eg-ui-gap-0 to .eg-ui-gap-12 gap mapped to spacing tokens
.eg-ui-grid-cols-1 to 12 grid-template-columns: repeat(N, minmax(0, 1fr));

Spacing & Sizing

Class Properties
.eg-ui-m-{0-12}, .eg-ui-p-{0-12} Margin/Padding all sides
.eg-ui-mx-{N}, .eg-ui-my-{N} Margin X (left/right) / Y (top/bottom)
.eg-ui-mt-{N}, .eg-ui-mr-{N}, etc. Margin Top/Right/Bottom/Left
.eg-ui-w-full, .eg-ui-w-auto width: 100%; / auto;
.eg-ui-h-full, .eg-ui-h-screen height: 100%; / 100vh;
.eg-ui-object-cover|contain|fill|none object-fit: cover|contain|fill|none;

Text Alignment & Overflow

Class Properties
.eg-ui-text-left|center|right|justify Text alignment
.eg-ui-text-truncate Truncates with ellipsis (...)
.eg-ui-overflow-auto|hidden|visible|scroll overflow property
.eg-ui-overflow-x|y-auto|hidden Axis-specific overflow

Z-Index & Order

Class Properties
.eg-ui-z-0 to 50 (step 10) z-index: 0, 10, 20...
.eg-ui-z-auto z-index: auto;
.eg-ui-order-1 to 6 Flex/Grid order
.eg-ui-order-first|last order: -9999 | 9999
.eg-ui-flex-shrink-0|shrink flex-shrink: 0 | 1
.eg-ui-flex-grow-0|grow flex-grow: 0 | 1
